# Employee-Tracker

## Description
The Employee Tracker is an application that I made that is a command-line tool developed using Node.js, MySQL, and Inquirer. It allows users to manage a company's employee database by providing a set of features for viewing, adding, and updating departments, roles, and employees.

## Installation
1. Clone the repository: git clone https://github.com/taycannon/Employee-Tracker
2. Navigate to the project directory: cd Employee-Tracker
3. Install dependencies: npm i inquirer@8.2.4
npm i mysql2
npm i express

## Usage

After starting the application, you will be presented with these list of options
Choose an option by using the arrow keys and pressing Enter

View All Departments
Add Department
View All Employees
Add Employee
View All Roles
Update Employee Role
Add Role
Quit
